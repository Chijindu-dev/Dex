## 🆘 Troubleshooting

### App doesn't show Bridge tab?
- Restart dev server: `npm run dev`
- Clear browser cache
- Hard refresh: Ctrl+Shift+R

### Bridge address shows undefined?
- Check .env file has VITE_BRIDGE_ADDRESS
- Restart dev server
- Check: `cat .env | grep VITE_BRIDGE`

### Fee looks wrong?
- Fee is 0.25% of amount
- Example: 100 tokens = 0.25 fee ✅

### Can't connect wallet?
- Install MetaMask
- Switch to Tempo testnet (42431)
- Connect wallet to app
